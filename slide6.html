玉手箱・GAB対策② 言語をロジカルに

VERBAL STRATEGY

GAB形式

論理的読解 / LOGICAL READING

01

本文全体をざっと読む

まずは大枠のテーマと主張を掴む。細部にとらわれすぎない。

02

設問を確認する

何を検証すべきかを把握し、本文の該当箇所を探す。

03

選択肢を検証（言い換え注意）

本文の言葉が微妙に言い換えられている「トラップ」に注意。

LOGIC CHECK / 判定基準

本文にある TRUE
本文と矛盾 FALSE
書かれていない UNKNOWN

IMAGES形式

趣旨判定 / GIST JUDGMENT

01

主張と根拠を見極める

「筆者が最も言いたいこと（主張）」と「それを支える事実（根拠）」を区別。

02

極端な表現に注意

「常に」「絶対に」「すべての」などの限定的表現は誤りの可能性大。

03

筆者の意図を捉える

本文に書かれていても、筆者の主張の本筋から外れるものは「趣旨」ではない。

INTENT CHECK / 趣旨判定

筆者の主張 YES
主張と矛盾 NO
関係なし IRRELEVANT

TARGET PACE / 目標ペース

長文1題あたり（設問4問程度）を読み解く目安時間。

60-90 SEC
